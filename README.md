# Bar Chart Visualization App

## Overview
US GDP visualized in a bar chart using D3.js.

## Features
- **Dynamic Data Visualization**: Utilizes D3.js for dynamic and interactive data visualization.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- D3.js (v7.8.5)
